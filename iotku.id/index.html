iotKu:

iotKu adalah platform Internet of Things dengan kode open-source

iotKu dibangun secara kolaboratif dari seluruh Indonesia, secara perlahan namun pasti. Teman-teman penggagas iotKu berharap dengan membangun iotKu memberikan kontribusi untuk Indonesia khususnya di bidang sumber terbuka dan IoT yang sedang booming.

Kenapa iotKu?

Mudah digunakan

iotKu mengutamakan kemudahan pengguna. Penggunaan API iotKu dibuat sangat sederhana sampai programmer pemula pun dapat menggunakannya tanpa kesulitan. Sangat cocok sebagai platform smart home.

Fleksibel

Pengguna dapat mendesain sendiri dashboard iotKu, karena dashboard merupakan client-side, bukan server-side. Ini membuat dashboard iotKu sangat fleksibel.

Universal

HTTP request digunakan untuk mengakses API iotKu. Ini membuat API iotKu dapat digunakan menggunakan device apapun, asalkan device tersebut dapat melakukan HTTP request.
